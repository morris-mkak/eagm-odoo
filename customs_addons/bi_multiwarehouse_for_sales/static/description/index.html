Sales line Multiple Warehouse Odoo Apps

Sale Order Multi Warehouse Odoo App helps users to select warehouses on each sale order line and delivery order created based on the warehouse selected on the sale order line. Users have the choice to first select the warehouse on product level and based on that automatically the warehouse will be filled on the sale order line afterwards users can still change to the auto selected warehouse. When a Sales order is confirmed multiple delivery orders are created according to the warehouse of the sales order line.

Features

Multi Warehouse for Sales Order

User can select multiple warehouse for sale orders.

Delivery Order Based on Warehouse

Delivery order will created based on warehouse selected on sale order line.

Change Warehouse on Sale Order

User can change or select different warehouse for sale order line.

Set warehouse for Products

User can set different warehouse for a different product and also this will appear on product variant.
Demo Request Video Tutorial

Multi Warehouse Configuration

Under Sales > Configurations > Settings menu user need to enable "Allow Warehouse in Sale Order Line".

Warehouse on Products

User have to select "Sale Warehouse" on product, Selected warehouse will also displayed on product variants.

Multi Warehouse on Sale Order

When user selects a products, related warehouse set on product will automatically set on sale order line, User can also change warehouse on sale order line.

Delivery Order Based on Warehouse

On confirming sale order user can see created delivery order for based on warehouse selected on sales order.

User can see all created transfer based on warehouse in tree view.

User can see created delivery order in form view for different warehouse.
