How to Setup

1. Sale Forecast Security/Access Rights

Only User with Sales Forecast Access rights have access to create the sales forecast records.

  • Go to settings.

Click on “User & companies" > Users.

Select the Full access and click on save.

2. Set a default Forecast Method

Before using the module you need to set a default forecast method. In order to do the same, Click on Sales Forecast menu

Once you click on “Sales Forecast” menu, you need to go to settings In order to do the same, Go to, Configuration > Settings

After clicking on “Settings” menu, Just select the forecast method and click on save.

3. Creating Sale Forecast Record

After installing the module you will be able to view that there is Sales Forecast Menu.

Click on the Create button to create a new forecast record.

Enter the "Start Date" and "End Date", select the forecast base and products. Enter the "Forecast Unit" and "Forecast Period" and "Forecasting Coffecients" after putting all the information you can click on “Save” button.

Using "Tuner Hyper Parameters" You can find the best forecasting coeffiecients from the range of provided values using Grid Search Algorithm. Best Values of all the Coefficients are vsible on Green div in top of form view.

Once you are done with creating the forecast record, just click on Predict Sales button

4. Visualization of Predicted Result

After clicking on Predict sales button. You can see the charts.

  • You can see the Line chart by default.

Chart visualization:

  • You can change it to bar chart.

Can also see the consolidate data is odoo default chart view.

Can also see the predicted values in pivot view.

5. How to change Forecast Method

Just click on change forecast method toggle button.

When you select Moving Average(MA) Method.

When you select Autoregressive Moving Average(ARMA) Method.

When you select Autoregressive Integrated Moving Average (ARIMA) Method.

When you select Seasonal Autoregressive Integrated Moving Average (SARIMA) Method.

When you select Vector Autoregression Moving-Average (VARMA) Method.

When you select "Autoregression (AR)", "Auto ARIMA", "Vector Autoregression (VAR)", "Simple Exponential Smoothing (SES)", "Holt Winter’s Exponential Smoothing (HWES)" Method.

6. Print Forecast Report

You Can Print Excel Report of Sales Forecast Records by "Print Forecast Report" Button.
