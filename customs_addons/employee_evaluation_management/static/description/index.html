The Perfect Employee Performance Evaluation Manager For Odoo

Create unique Survey for Each designation / Position

  • Employee is different and each job position is so Odoo employee Evaluation system allows you to create separate assessment questions groups for each position.
  • Store the questions in Odoo and use them to create performance reports to get feedback.
  • Each question can be of a different answer type making it easier to get desired feedback from managers.

Keep it fair and transparent

  • Since the questions cannot be altered in a report, employees with the same job position are assessed with the same questions.
  • Same questions for the same position increases transparency and curbs chances of any biased feedback.

Digitize the Records

  • All the evaluated reports are digitally stored in Odoo.
  • It makes it more efficient and organized than manual reports and feedback.
  • They can used later at the time appraisal, analysis and in other important decisions.

Detailed Features List

Below is the detailed list of Feature for Odoo Employee Evaluation Management.

Install Employee Performance System in Odoo

  • The module adds a complete employee evaluation manager in your Odoo.
  • You can now access and get feedback on performance reports from managers.
  • Note: The employee needs to have at least 'User' Access in the project to see the evaluation menu at his end.
  • Note: To create question templates for employees the user needs to have 'Manager' access in the project.
  • You can manage these settings from user access rights.

Create Your Own Assessment questions For Each Job Position

  • You can add feedback questions templates according to different job positions.
  • Each job position can have questions set to be used in evaluation reports.
  • There are 7 answer input types to choose from for each question.
  • For example; multiple choice, text type answer, Datetime, numerical, etc.

Create Odoo Evaluation report for each employee

  • The Employee can create an evaluation report for himself.
  • The manager can create evaluation reports for his team.
  • The report can be created for the desired time-period like weekly, monthly, etc.
  • The questions are automatically added to the report after selecting the job designation.

Shows work hour summary for employee in Evaluation report

  • The report automatically shows the employee's working hour summary for the selected period.
  • This helps the manager in proper cross-checking while answering the questions.

Manager Control on Reports Approval

  • The Employees can only create and submit their report. They cannot answer the questions.
  • Only the manager can approve the evaluation report.
  • The Manager needs to answer all the questions in the employee evaluation report before approving it.

Project Manager Control on Reports Done State

  • Once reports are approved, only the project manager can put the reports to Done status.

The admin can create employee evaluation templates according to job positions

The admin can enter questions and answer type for managers in each template

Create evaluation report for employees for desired time-period

  • The manager can create report for any employee.
  • The employee can create the report for himself only.

    • The question template is automatically loaded as per the position of selected employee in the report

    Confirming the evaluation report

  • Employees cannot confirm their reports.
  • Only the employee's manager has the access to do so. The manager needs to answer the questions in each report before confirming it.

    • The report can be put to done stage only by the project manager
